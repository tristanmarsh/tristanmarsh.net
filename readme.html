Deployment

Deployment of a WordPress installation requires that the databse and wp-config.php be refactored from the old domain to the new domain. Refactoring the database is best accomplished using the interconnectit SRDB tool.

A standard WordPress installation merely requires the database details to be updated accordingly. WordPress multisite or installations with certain wp-config modifications require domain updates also.
